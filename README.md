🛍️ #Shopping Behaviour Analysis | EDA + PCA + K-Means Clustering

📌 Project Overview

This project focuses on analyzing customer shopping behaviour using Exploratory Data Analysis (EDA), Principal Component Analysis (PCA), and K-Means Clustering techniques.

The objective of this project is to identify hidden customer segments, understand purchasing patterns, and derive meaningful business insights from the dataset.

🚀 #Features

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization using Seaborn & Matplotlib
Feature Engineering
Dimensionality Reduction using PCA
Customer Segmentation using K-Means Clustering
Cluster Visualization & Insights

🧠 #Technologies Used

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

📊 Project Workflow

1️⃣ Data Preprocessing
Handling missing values
Encoding categorical variables
Feature scaling
Removing unnecessary columns

2️⃣ Exploratory Data Analysis (EDA)

Performed detailed visual analysis to understand:

Customer demographics
Purchase patterns
Category preferences
Spending behaviour
Correlation between features

Visualizations used:

Count plots
Bar plots
Heatmaps
Histograms
Scatter plots

3️⃣ Principal Component Analysis (PCA)

Applied PCA for:

Reducing dimensionality
Removing redundancy
Improving clustering efficiency
Better visualization of high-dimensional data

High Dimensional Data→Principal Components

4️⃣ K-Means Clustering

Implemented K-Means clustering to segment customers into different groups based on shopping behaviour.

Steps performed:

Optimal cluster selection using Elbow Method
Model training using K-Means
Cluster assignment
Cluster visualization

📈 Key Insights

Identified distinct customer segments
Analyzed purchasing trends across clusters
Discovered behavioural similarities among users
Generated insights useful for targeted marketing and recommendation systems
